What is Arquillian OSGi?

ShrinkWrap is the simplest way to create archives in Java. Using the fluent and intuitive ShrinkWrap API, developers may assemble JARs, WARs, and EARs to be deployed directly by Arquillian during testing.

Artifacts

Build dependency artifacts

Add the appropriate stanza to the <dependencies> element in your pom.xml:

Arquillian OSGi :: Container :: JBoss :: Embedded

<dependency>
  <groupId>org.jboss.arquillian.osgi</groupId>
  <artifactId>arquillian-container-jbosgi-embedded</artifactId>
  <version>2.2.1.Final</version>
  <scope>test</scope>
</dependency>
Warning: You'll also need to add the Arquillian core platform and possibly additional dependencies. Please refer to the guides for more details.
