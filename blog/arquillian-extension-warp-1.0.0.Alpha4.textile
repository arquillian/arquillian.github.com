I have decided to release another Alpha release since it takes longer than expected to get through "issues schedule for Beta":https://issues.jboss.org/browse/ARQ/fixforversion/12320886.

This release is not heavy in terms of number of fixes, but still it's important one:

h3. 



h3. Highlighted Changes


- <i class="icon-star"></i> EAR Deployments Support :=
Thanks to great contribution from "Robert Panzer":https://github.com/robertpanzer, Warp now supports EAR Deployments! =:

- <i class="icon-star"></i> Critical Bug Fix :=
With this release, you can leverage a fix for @The response payload with serialId 4065404507533364875 was never registered@ issue. It not only caused some requests fail at all, but it made some tests non-deterministic and intermitently failing.
=:

h3. Road to Beta...

Since the road to the Beta can still take some time, we are going to do more Alpha releases in timely manner, so you won't lose touch with latest bug fixes, especially if have you contributed them!
