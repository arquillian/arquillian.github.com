Arquillian Droidium 1.0.0.Alpha5 is packed with bugfixes, updates and it has integrated with various Arquillian extensions.

Regarding of extension integration, Droidium now profits from Arquillian Spacelift extension which provides robust way how to deal with execution of external binaries and programs. Being Droidium-specific - it deals with execution of all Android-related binaries.

Other extension which is from 1.0.0.Alpha5 integrated with Droidium is Arquillian Droidium Screenshooter extension which builds on top of Arquillian Reporting extension. Having Arquillian reporting extension on class path, you can make reports of your (mobile) test very comfortably in a lot of possible export formats like XML, JSON, HTML or even Asciidoctor.

Arquillian Droidium 1.0.0.Alpha5 is the last release before reaching its Beta stage. Starting with Beta1, you can expect experimental support for Arquillian Drone 2.0.0.Alpha1, removing of web pluging which depends on deprecated Selenium Android server APK and depending on Selendroid's Android driver. There will be also an update to Selenium 2.40 from current 2.39 where Android driver is deprecated.
