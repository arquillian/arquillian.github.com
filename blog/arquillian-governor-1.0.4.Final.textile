We are proud to announce new features in this release for Arquillian Governor:

h3. New Features

- <i class="icon-star"></i> Arquillian Recorder integration with Arquillian Governor:=

p. In this release we are providing recorder integration with Jira, GitHub & Redmine Governor.

p. Arquillian Recorder project brings neat reports of your Arquillian tests. Now we are providing an integration to the project so that generated report contains information about JIRA, GitHub and Redmine properties such as Issue URL, Detector & it's strategy used for environment, force value.

p.You want to use recorder with JIRA governor, you only need to import recorder and @governor-jira@ dependencies:

bc(prettify).. <dependency>
  <groupId>org.arquillian.extension</groupId>
  <artifactId>arquillian-recorder-reporter-impl</artifactId>
  <version>${version.arquillian.recorder}</version>
  <scope>test</scope>
</dependency>
<dependency>
    <groupId>org.arquillian.extension</groupId>
    <artifactId>arquillian-governor-jira</artifactId>
    <version>${version.jira.governor}</version>
    <scope>test</scope>
</dependency>

p.Same way, if you want to use recorder with GitHub governor, you only need to import recorder and @governor-github@ dependencies:

bc(prettify).. <dependency>
  <groupId>org.arquillian.extension</groupId>
  <artifactId>arquillian-recorder-reporter-impl</artifactId>
  <version>${version.arquillian.recorder}</version>
  <scope>test</scope>
</dependency>
<dependency>
    <groupId>org.arquillian.extension</groupId>
    <artifactId>arquillian-governor-github</artifactId>
    <version>${version.github.governor}</version>
    <scope>test</scope>
</dependency>

p.To use recorder with Redmine governor, you only need to import recorder and @governor-redmine@ dependencies:

bc(prettify).. <dependency>
  <groupId>org.arquillian.extension</groupId>
  <artifactId>arquillian-recorder-reporter-impl</artifactId>
  <version>${version.arquillian.recorder}</version>
  <scope>test</scope>
</dependency>
<dependency>
    <groupId>org.arquillian.extension</groupId>
    <artifactId>arquillian-governor-redmine</artifactId>
    <version>${version.redmine.governor}</version>
    <scope>test</scope>
</dependency>


p. You can read more about Arquillian Recorder integration with Jira Governor in the "Arquillian Recorder Integration Section with Jira":https://github.com/arquillian/arquillian-governor#arquillian-recorder-integration-with-arquillian-jira-governor-extension, GitHub Governor in the "Arquillian Recorder Integration section with GitHub":https://github.com/arquillian/arquillian-governor#arquillian-recorder-integration-with-arquillian-github-governor-extension, Redmine Governor in the "Arquillian Recorder Integration section with Redmine":https://github.com/arquillian/arquillian-governor#arquillian-recorder-integration-with-arquillian-redmine-governor-extension.
